  <Steps>

    <Note>
    **Documents de vérification (KYC)** 
    
    En fonction de votre type de compte, certaines informations sont nécessaires :

      - **ONG** : Ce compte est destiné aux organisations non gouvernementales. Aucun plafond de montant pour les transactions.

      - **Travailleur indépendant** : Conçu pour les freelances, ce compte permet un maximum de 10 transactions par semaine, limitées à un montant entre 100 XOF et 300 000 XOF par transaction.

      _**Documents requis** : Carte d’identité, IFU, signature._

      - **Business** : Pour les entreprises enregistrées formellement, sans limite de montant pour les transactions.

      _**Documents requis** : RCCM, IFU, carte d’identité, signature._

      - **Gouvernement** : Compte destiné aux institutions gouvernementales, sans plafond de montant.
    </Note>

### Scénario 1: Recevoir des paiements rapidement avec un lien de paiement

Si vous avez besoin de recevoir de l’argent de vos clients sans intégration technique complexe, **FedaPay** est votre solution. Il vous permet de générer des liens de paiement sécurisés en quelques minutes, que vous pouvez envoyer directement à vos clients.

**3 étapes simples** :

<Steps>
  <Step title="Créer un client">
    - Depuis votre tableau de bord FedaPay, renseignez les informations de base du client : **Nom, Prénom, Email** (l'email est unique et essentiel pour suivre vos clients).
    - Le **numéro de téléphone** est facultatif.
  </Step>
  <Step title="Créer une transaction">
    - Une transaction représente le paiement que vous attendez du client. Il vous suffit d’indiquer le **montant**, **une description** (obligatoire) et, si vous le souhaitez, un lien de retour personnalisé.

  </Step>
  <Step title=" Générer un lien de paiement">
    - Après avoir créé la transaction, un lien de paiement sécurisé sera généré. Ce lien peut être envoyé au client par email ou tout autre moyen de communication (WhatsApp, SMS, etc.).
    - Ce lien est **valable pendant 24 heures** et garantit une transaction sécurisée et validée par FedaPay.

  </Step>
</Steps>


### Scénario 2: Intégrer FedaPay à votre plateforme e-commerce

Pour les entreprises qui souhaitent accepter des paiements directement sur leur site web ou application mobile, **FedaPay** offre une solution clé en main. Vous pouvez rapidement intégrer un formulaire de paiement sécurisé avec notre module **Feda Checkout**, qui combine HTML, JavaScript et CSS.

**3 étapes pour intégrer FedaCommerce :**

<Steps>
  <Step title="Tester Feda Checkout">
    - Faites un test rapide pour voir comment le processus fonctionne sur votre site. Remplissez un formulaire de paiement de test avec des informations comme un **nom**, **prénom** et **email** valide.
  </Step>

  <Step title="Ajout d’un bouton de test ici pour une meilleure expérience utilisateur.">
  </Step>

  <Step title="Intégrer le code de paiement">
    - Ajoutez le code d'intégration à votre page de paiement. Ce code est conçu pour être simple à intégrer et sécurisé.
    - Assurez-vous de remplacer **VOTRE_CLE_API_PUBLIQUE** par votre clé API publique que vous trouverez dans votre compte FedaPay.

    <Note>
  _**Conseil** : Utilisez toujours la clé API publique pour les intégrations côté client et gardez la clé API privée pour votre serveur._
  </Note>

  </Step>

  <Step title="Connecter votre compte à Feda Checkout">
    - Une fois le code intégré, vous devrez connecter votre compte FedaPay à Feda Checkout pour que votre site puisse traiter les paiements.
    - Cette étape est simple et ne doit être effectuée qu'une seule fois. Une fois votre compte lié, vos utilisateurs pourront effectuer des paiements en un clic depuis votre site ou application.

  </Step>
</Steps>

## Création d’un compte

#### Créer un compte FedaPay : Mode Test et Mode Live

Commencer avec FedaPay est simple et rapide. Vous avez la possibilité de créer deux types de comptes : **Test (Sandbox)** et **Live**.

1. **Compte Test (Sandbox)** : Ce compte vous permet de tester toutes les fonctionnalités de FedaPay sans réaliser de transactions réelles. Aucune validation n'est requise, et vous pouvez explorer la plateforme à votre rythme.
2. **Compte Live** : Ce compte est nécessaire pour accepter de vrais paiements. Une fois que vous avez créé un compte Live, vous devrez soumettre certains documents pour validation avant de pouvoir l'utiliser pleinement.

### Étapes pour créer et activer votre compte

<Tabs>
  <Tab title="Créer un compte FedaPay (Test ou Live)">
    - [Accédez au site FedaPay](https://www.fedapay.com/?v=ddd70bbaf5be) et cliquez sur **"S'inscrire"**.
    - **Remplissez le formulaire d'inscription** avec vos informations de base (nom, email, etc.).
    - Vous pouvez commencer en mode **Test** immédiatement après votre inscription.

    ![Formulaire d'inscription ou de connexion](https://res.cloudinary.com/dvilp6td2/image/upload/v1730221481/Screenshot_of_FedaPay_okjarq.jpg)

  </Tab>

  <Tab title="Activer votre compte">
    Pour passer du compte Test au compte Live, vous devez activer votre compte :
      - **Confirmez votre email** : Après votre inscription, vérifiez votre boîte de réception et cliquez sur le lien de confirmation que nous vous envoyons.
      - **Soumettez des documents** : Pour un compte Live, certaines informations supplémentaires sont requises selon votre statut (particulier, entreprise, ONG).

    Voici les documents requis pour l'activation du compte Live

      <Steps>
        <Step title="Compte Individuel/Startup">
          - Une copie de votre carte d’identité ou passeport.
          - Une preuve de votre activité (facultatif pour les startups).
        </Step>

        <Step title="Compte ONG">
          - Une copie de la pièce d’identité du fondateur ou du gérant.
          - Le numéro d’enregistrement de l’ONG et une copie du document officiel.

        </Step>

        <Step title="Compte Société">
          - La pièce d'identité du gérant.
          - Le numéro IFU/TVA et le numéro RCCM avec les documents correspondants.

        </Step>
      </Steps>


  </Tab>

  <Tab title="Faites valider votre compte">
    Une fois que vous avez réuni les documents nécessaires, suivez ces étapes :
    - **Connectez-vous** à votre tableau de bord FedaPay.
    - Cliquez sur **"Activer mon compte"** ou accédez aux paramètres de votre compte.
    - **Remplissez le formulaire de validation** : 
    Indiquez le nom de votre entreprise, son domaine d'activité, et ajoutez les documents requis.
    - Cliquez sur **"Soumettre"** pour finaliser la demande.

    <Info>Illustration : Formulaire d'activation du compte (image montrant le formulaire rempli)</Info>

    Une fois la demande envoyée, l’équipe FedaPay examinera les informations et les documents fournis. Vous recevrez une notification pour vous informer de l’état de votre demande :

    - **Si le compte est validé**, cette information apparaîtra sur votre tableau de bord, indiquant que le compte est actif et prêt à l’emploi.
    - **Si des informations manquent ou que des documents supplémentaires sont requis**, FedaPay vous contactera pour les fournir afin de finaliser la validation.

    Une fois approuvé, le statut de votre compte sur le tableau de bord affichera qu’il est **validé**.
  </Tab>
</Tabs>

<Note>
Avant que votre compte ne soit validé, vous aurez accès à des fonctionnalités limitées :
- **Paiements uniquement par Mobile Money.**
- **5 transactions maximum par semaine.**
- **Montant par transaction** : entre 100 XOF et 5000 XOF.

Après validation, vous accéderez à l'intégralité des fonctionnalités de FedaPay.
</Note>

### Sécuriser votre compte

La sécurité est primordiale chez FedaPay. Voici quelques conseils pour protéger votre compte :

- **Utiliser un accès limité pour vos collaborateurs** : Vous pouvez ajouter des utilisateurs avec des autorisations spécifiques pour gérer des parties de votre compte.
- **Mettez à jour régulièrement votre ordinateur et votre navigateur** pour éviter les vulnérabilités.
- **Méfiez-vous du phishing** : Ne partagez jamais votre mot de passe après avoir cliqué sur des liens non sécurisés. FedaPay utilise toujours des liens HTTPS et le domaine officiel fedapay.com .

Créer et activer votre compte FedaPay est un processus simple et sécurisé. Une fois validé, vous pourrez accepter des paiements en toute confiance, que vous soyez une startup, une ONG ou une grande entreprise.
