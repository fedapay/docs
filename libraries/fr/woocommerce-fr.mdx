---
title: "WooCommerce"
---

Intégrer FedaPay à votre boutique WooCommerce est simple grâce à notre plugin dédié pour WordPress. Suivez ce guide pour installer et configurer le plugin afin de commencer à accepter des paiements en ligne rapidement.

<Steps>
  <Step title="Télécharger et installer le plugin">
    Vous pouvez télécharger le plugin FedaPay :

    * Directement depuis ce [Lien](https://wordpress.org/plugins/woo-gateway-fedapay/)

    * Ou depuis votre tableau de bord WordPress, en allant dans **Extensions** > **Ajouter** et en recherchant **"FedaPay WooCommerce"**.

    ![Image montrant le plugin dans WordPress](https://res.cloudinary.com/dvilp6td2/image/upload/v1731408081/plugin-WordPress_pv4rcu.png)
  </Step>

  <Step title="Activer le plugin">
    Une fois le plugin installé, activez-le en cliquant sur **Activer**. Ensuite, dirigez-vous vers les paramètres de WooCommerce pour finaliser la configuration.
  </Step>

  <Step title="Accéder aux paramètres de WooCommerce">
    Pour configurer FedaPay, allez dans le tableau de bord WordPress puis cliquez sur :

    * **WooCommerce**

    * **Réglages**

    * **Paiements**

    Dans cette section, vous verrez tous les moyens de paiement disponibles. Tout en bas, vous trouverez **FedaPay**.

    ![Image montrant les paramètres de votre boutique WooCommerce](https://res.cloudinary.com/dvilp6td2/image/upload/v1731408081/reglage-woocommerce_trdani.png)
  </Step>

  <Step title="Configurer la passerelle FedaPay">
    Cliquez sur **Configuration** à côté de FedaPay pour ouvrir les options de configuration.

    Voici les principaux paramètres à configurer :

    * **Activer FedaPay**: Cochez cette option pour activer FedaPay et recevoir des paiements en direct.

    * **Mode Test (sandbox)** : Cochez cette case uniquement si vous effectuez des tests. Laissez-la décochée lorsque vous passez en mode live pour accepter les paiements réels.

    ![Image montrant l'interface de configuration de FedaPay](https://res.cloudinary.com/dvilp6td2/image/upload/v1731408081/configuration-woocommerce-fedapay_aa6k5c.png)
  </Step>

  <Step title="Récupérer et entrer vos clés API">
    Pour utiliser FedaPay, vous devez entrer vos clés API (privées) pour les environnements test et live. Voici comment les récupérer :

    1. Connectez-vous à votre compte FedaPay.

    2. Accédez à votre **tableau de bord** FedaPay.

    3. Trouvez les clés API de vos comptes **Sandbox** (test) et **Live** (production).

       * La clé privée pour le mode **live** commence par **sk\_live**.

       * La clé privée pour le mode **sandbox** (test) commence par **sk\_sandbox**.

    4. Copiez ces clés et collez-les dans les champs correspondants dans les paramètres WooCommerce.

    ![Image montrant les clés API](https://res.cloudinary.com/dvilp6td2/image/upload/v1731408081/cle-woocommerce-fedapay_nwyzog.png)
  </Step>

  <Step title="Enregistrer les modifications">
    Une fois les clés API correctement entrées, cliquez sur **Enregistrer les modifications** pour finaliser la configuration.
  </Step>

  <Step title="Passerelle de paiement prête">
    Félicitations, votre passerelle FedaPay est maintenant configurée et prête à être utilisée pour accepter les paiements de vos clients directement depuis votre boutique WooCommerce !

    ![Image montrant la passerelle de paiement active dans WooCommerce](https://res.cloudinary.com/dvilp6td2/image/upload/v1731408081/passerelle-activer_ekwa2d.png)
  </Step>
</Steps>

Avec ces étapes simples, vous pouvez maintenant profiter des paiements en ligne rapides et sécurisés grâce à FedaPay sur votre boutique WooCommerce.
